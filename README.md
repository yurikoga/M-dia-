# Neumorphism Effect on Social Media Icons CSS | Neumorphism Hover Animation | CodingLab

Um projeto simples de **ícones de redes sociais com efeito neumorphism** usando HTML e CSS.

## ✨ Sobre

Este projeto demonstra como criar **ícones com efeito neumorphism e animação no hover**, deixando o design mais moderno e elegante.

## 🛠️ Tecnologias

- HTML
- CSS (box-shadow, hover effects)

## 🎯 Funcionalidades

- Efeito neumorphism
- Animação ao passar o mouse
- Design moderno e minimalista

## 🚀 Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
